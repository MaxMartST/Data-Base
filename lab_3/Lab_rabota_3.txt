0. Выбор СУБД

Необходимо выбрать СУБД, с которой вы будете работать в течении семестра.
Наиболее популярные решения:
	MYSQL https://www.mysql.com/
	MSSQL https://www.microsoft.com/ru-ru/sql-server/sql-server-editions-express
	PostgreSQL https://www.postgresql.org/
    
Также разрешается выбрать другую реляционную СУБД.
Преимущества: 
		
	- MSSQL СУБД с большим набором вспомогательный функций и утилит. 
		Много информации для исследования как работает СУБД и как выполняются запросы. Используется в компании Travelline.
		Есть графический интерфейс SQL Server Management Studio

    - PostgreSQL современная СУБД с богатым функционалом. Используется в компании OmegaR.

    - MySQL популярная СУБД, но не самый лучший выбор. Есть графические инструменты PHPMyAdmin, MySQL Workbench. Используется в компании iSpring.
		Можно установить в пакете WAMP (PHP, MySQL, Apache...)

Как установить можно посмотреть в интернете, например, на youtube.

Рекомендую создать базу данных в облачной платформе: Microsoft Azure, Google Cloud Platform, Amazon Web Services
